PREREQUISITES
=============
- Install Node.js from www.nodejs.org


CONFIGURE
=========
Edit the file index.js and set the values to :

-URL_API
-USERNAME
-PASSWORD
-APP_KEY
-APP_SECRET


RUNNING
=======
- Install node modules
- Run 'npm install' (On linux, you need 'sudo' in front of the command)
- Then run 'node index.js'
